SEO

Search Engine Optimization 簡單來說，當使用者使用關鍵字在搜尋引擎進行搜尋時，使網頁能夠出現在結果排序前面的方法。

How To (依影響度排序)

  1. 網站流量
  2. 社群網站分享與參與度
    • Facebook Like and Share
    • Tweet
    • etc..
  3. title
    • 越相關的字，在 title 往越前面擺，總長不要超過 20-25 個字
  4. Sitemap
  5. 關鍵字在網頁上的密度
  6. URL 優化
    • Clean URL
    • Slugs
    • 英文使用小寫，單字間使用 - 分隔
  7. Html 架構正確
    • 使用 Html 5 架構撰寫頁面，HTML 5 Semantics
    • 參考 Schema.org 撰寫 tag name

已不被參考的 tag

  • keywords
  • description (還是很重要，關係到在搜尋結果頁面出現的字，影響使用者點擊意願)

Useful Tools

  • 關鍵字
  • Google AdWords API
  • Google Analytics
  • 搜尋索引註冊、網站優化
  • Google WebMasters
  • Solution
  • Moz
