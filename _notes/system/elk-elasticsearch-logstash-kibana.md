## Kibana

### Install

```bash
/opt/kibana
```

deamon <https://github.com/akabdog/scripts/blob/master/kibana4_init>

```bash
/etc/init.d/kibana
```

```bash
/etc/init.d/kibana start
```

start on boot

```bash
update-rc.d kibana defaults
```
